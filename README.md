# 🍕 Pizza Sales Analysis using SQL & Power BI

This is my **Pizza Sales Analysis** project. In this project, I used **SQL** and **Power BI** to analyze pizza sales data and create an interactive dashboard.

The dashboard shows **Total Sales**, **Pizza Size**, **Pizza Price**, and **Quantity Sold**. It also shows the sales trends for **June** and **July**.

## **SQL**

- Imported the **raw data** from an **Excel** file into **SQL Server**.
- Cleaned and processed the data before analysis.
- Wrote SQL queries based on the dashboard requirements.
- Used **JOINs**, **Aggregate Functions (SUM, COUNT, AVG)**, **GROUP BY**, and **ORDER BY** to generate business insights.

## **Power BI**

- Connected **Power BI** to **SQL Server** using the SQL Server connector.
- Performed the **ETL (Extract, Transform, Load)** process:
  - **Extracted** data from **SQL Server**.
  - **Transformed** the data using **Power Query** by cleaning and preparing it for analysis.
  - **Loaded** the transformed data into **Power BI**.
- Created an interactive dashboard using different visualizations.
- Added interactive filters and slicers for better data analysis.

## **Dashboard Features**

- 📊 **Pie Chart** to show sales by **Pizza Category**.
- 📈 **Bar Chart** and **Area Chart** to show sales trends and comparisons.
- 🔄 **Toggle Button** to switch between **Best Seller** and **Worst Seller** pizzas.
- 📅 Sales analysis for **June** and **July**.
- 📌 Displays **Total Sales**, **Pizza Price**, **Pizza Size**, and **Quantity Sold**.

## **Tools Used**

- **SQL Server**
- **Power BI**
